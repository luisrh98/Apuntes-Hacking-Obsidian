# Definición

**Drupal** es una plataforma de gestión de contenidos (CMS) de código abierto que permite crear, actualizar y gestionar contenidos de un sitio web de manera flexible y dinámica. Está diseñado para ser un sistema multipropósito, lo que significa que puede utilizarse para desarrollar cualquier tipo de proyecto web, desde sitios institucionales básicos hasta sitios complejos con funcionalidades específicas.

Drupal utiliza tecnologías como **PHP, MySQL, PostgreSQL o SQLite** como base de datos.
