# Definición
>**WordPress** es un [sistema de gestión de contenidos](https://es.wikipedia.org/wiki/Sistema_de_gesti%C3%B3n_de_contenidos "Sistema de gestión de contenidos"), enfocado a la creación de cualquier tipo de [página web](https://es.wikipedia.org/wiki/P%C3%A1gina_web "Página web"). Originalmente alcanzó una gran popularidad en la creación de [blogs](https://es.wikipedia.org/wiki/Blogs "Blogs"), para luego convertirse en una de las principales herramientas para la creación de páginas web comerciales.
>WordPress está desarrollado en el lenguaje [PHP](https://es.wikipedia.org/wiki/PHP "PHP") para entornos que ejecuten [MySQL](https://es.wikipedia.org/wiki/MySQL "MySQL"), [MariaDB](https://es.wikipedia.org/wiki/MariaDB "MariaDB") y [Apache](https://es.wikipedia.org/wiki/Servidor_HTTP_Apache "Servidor HTTP Apache"), bajo licencia [GPL](https://es.wikipedia.org/wiki/GNU_General_Public_License "GNU General Public License") y es [software libre](https://es.wikipedia.org/wiki/Software_libre).
