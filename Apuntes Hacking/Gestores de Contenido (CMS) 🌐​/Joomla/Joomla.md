# Definición

Joomla es un sistema de gestión de contenidos (CMS) de código abierto que permite crear y administrar sitios web de manera sencilla. Está construido sobre el lenguaje de programación **PHP** y utiliza bases de datos **MySQL o PostgreSQL** para almacenar y gestionar el contenido. Además, utiliza tecnologías web estándar como **HTML, CSS y JavaScript** para la presentación y la interacción en la interfaz de usuario. También incorpora el marco de trabajo **MVC** (Modelo-Vista-Controlador)
