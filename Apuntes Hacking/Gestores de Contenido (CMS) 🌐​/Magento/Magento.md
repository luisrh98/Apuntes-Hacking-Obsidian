# Definición

**Magento** es una plataforma de comercio electrónico de código abierto desarrollada en PHP, que permite crear y gestionar tiendas online. Utiliza el sistema de base de datos relacional MySQL/MariaDB y elementos del framework Zend. Además, emplea la programación orientada a objetos y la arquitectura modelo-vista-controlador.
