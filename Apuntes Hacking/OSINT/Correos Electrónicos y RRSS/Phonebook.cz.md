# Definición

[[Phonebook.cz]] es un servicio de agregación de datos que recopila y compara información de diversas fuentes, incluyendo datos filtrados de malware o brechas de seguridad. Este sitio permite a los usuarios buscar dominios, direcciones de correo electrónico o URLs, y actualmente cuenta con 165 mil millones de registros.
