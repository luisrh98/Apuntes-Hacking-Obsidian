# Definición

Un [[payload staged]] es un tipo de payload utilizado en Metasploit que se ejecuta en fases o etapas. Este tipo de payload se divide en partes, donde una parte inicial, conocida como stager, se encarga de establecer una conexión con el atacante y descargar el resto del payload, que contiene las funciones específicas que se deben ejecutar en el sistema objetivo. El stager tiene un tamaño mínimo y es más inocuo para el sistema del objetivo, mientras que el resto de las fases (stages) se encargan de realizar las tareas específicas, como obtener una shell o una consola de **Meterpreter**.
